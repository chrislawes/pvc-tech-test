### 🚀 Registration page

A Nuxt 3 SPA registration page that allows new users to sign up with their email and create a secure password.

### 🛠️ Built using

- Vue / Nuxt 3
- Zod validation
- Playwright
- Vitest

### 🏃‍♂️ Run locally

```shell
npm i
npm run dev
```

### 🧪 Testing

```shell
npm run test:e2e
```

```shell
npm run test:unit
```
